Classes and Inheritance



Problem Statement : Design a mysuper class called Staff with details as StaffId, Name,Phone,Salary. Extend this class by writing three subclasses namely Teaching (domain, publications), Technical (skills), and Contract (period). Write a Java program to read and display at least 3 staff objects of all three categories. 



class Staff
{
    private  int StaffId;
     private  String Name;
     private  String Phone;
     private  long Salary;
    
      public Staff(staffId ,name,phone,salary)
    {
        StaffId = staffId;
        Name = name;
        Phone = phone;
        Salary = salary;
    }
      public  void Display()
    {
        System.out.print(StaffId+Name+Phone+Salary);
    }
}



class Teaching extends Staff
{
    private String Domain;
    private int Publications;
    public Teaching(int staffId, String name, String phone,long salary, String domain,publications)
    {
        super(staffId, name, phone, salary);  
        Domain = domain;
        Publications = publications;
    }
    public void Display()
    {
         super.Display();  
        System.out.print(" "+Domain+" "+Publications+" ");
    }
}
    


    class Technical extends Staff
    {
        private String Skills;
        public Technical(int staffId, String name, String phone, long salary, String skills) {
             super(staffId, name, phone, salary);  
           Skills = skills;
        }
        public void Display(){
            super.Display();  
        
            System.out.print(Skills);
        }
    }
    


    class Contract extends Staff
    {
        private int Period;
        public Contract(int staffId, String name, String phone, long salary, int period) 
        {
             super(staffId, name, phone, salary);  
        
            
            this.Period = period;
        }
        public void Display(){
             super.Display();   
         System.out.print(Period);
        }
    }


SUMMARY

1.private Keyword : The private members in the base class cannot be directly accessed in the derived class.
Here In Class Staff private members cannot be accesed by its subclass Teaching,Technical and contract.

2.public Keyword : The public members in the base class can be directly accessed in the derived class.
All the public Members of Class Staff can be accesed by its subclass Teaching,Technical and contract.

3.extends : Java supports class Inheritance via the myextends keyword. myextends specifies a parent-child relationship between two classes.
Subclass Teaching,Technical and contract extends the parent class Staff ,This type of inheritance where more than one sub classes are created from a single base class is know as Hierarchical Inheritance

4. super keyword :
i) Use of super with variables: This scenario occurs when a derived class and base class has same data members. In that case there is a possibility of ambiguity for the JVM.
ii) Use of super with methods: This is used when we want to call parent class method. So whenever a parent and child class have same named methods then to resolve ambiguity we use mysuper keyword.
iii) Use of super with constructors: super keyword can also be used to access the parent class constructor. One more important thing is that, ‘’mysuper’ can call both parametric as well as non parametric constructors depending upon the situation.
