Quick Sort | Sorting



Problem Statement : QuickSort is a Divide and Conquer algorithm. It picks an element as pivot and partitions the given array around the picked pivot. There are many different versions of quickSort that pick pivot in different ways



Quick Sort | Sorting

Quick Sort is an algorithm of Divide & Conquer type


Divide: Rearrange the elements and split arrays into two sub-arrays and an element in between search that each element in left sub array is less than or equal to the average element and each element in the right sub- array is larger than the middle element.

Conquer: Recursively, sort two sub arrays.

Combine: Combine the already sorted array.




Algorithm 

QUICKSORT (array A, int m, int n)   
 1 if (n > m)   
 2 then   
 3 i ← a random index from [m,n]   
 4 swap A [i] with A[m]   
 5 o ← PARTITION (A, m, n)   
 6 QUICKSORT (A, m, o - 1)  
 7 QUICKSORT (A, o + 1, n)

Partition Algorithm

PARTITION (array A, int m, int n)   
 1 x ← A[m]   
 2 o ← m   
 3 for p ← m + 1 to n   
 4 do if (A[p] < x)   
 5 then o ← o + 1    
 6 swap A[o] with A[p]  
 7 swap A[m] with A[o] 
 8 return o

Parition Function

We start with two variables i and j. Out of which, j initially points to first element and i points to the previous index of the first element. We start comparing a[ j ] with pivot/last element and if a[ j ] <= pivot then increment i and swap a[ i ] with a[ j ]. It should be noted that j is from p t0 r-1, incrementing by 1 each time; where array is a[p, p+1 ... r].



In the above illustration, we can observe that elements before i is always smaller than pivot at any given time and elements between i and j is greater or equal to pivot at any given time.
